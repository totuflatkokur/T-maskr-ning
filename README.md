# Tótu Stimpilklukka v2

Hreint app — bara stimpilklukka.

## Setja upp
1. Keyrðu `supabase_stimpilklukka_v2.sql` í Supabase SQL Editor.
2. Opnaðu `supabase.js`.
3. Skiptu út `SETTU_PUBLISHABLE_KEY_HER` fyrir publishable key úr Supabase.
4. Settu skrárnar í GitHub:
   - index.html
   - style.css
   - app.js
   - supabase.js
   - manifest.json
5. Commit changes.
6. Netlify deployar sjálfkrafa.
